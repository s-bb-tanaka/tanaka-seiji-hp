# 田中せいじ オフィシャルサイト

「現場主義。声を聞きに行く政治。」をコンセプトにした、田中せいじ氏の政治活動オフィシャルサイトです。

## ファイル構成

```
tanaka-seiji-hp/
├── index.html          ← サイト本体(HTML/CSS/JS 全部入り、1ファイル完結)
├── assets/
│   ├── portrait.svg    ← ヒーロー部の顔写真プレースホルダー
│   ├── profile.svg     ← プロフィール欄の活動写真プレースホルダー
│   └── favicon.svg     ← ブラウザタブのアイコン
└── README.md           ← このファイル
```

## サイト構成(セクション)

1. **HERO** — キャッチコピー「現場主義。」とサブ「声を聞きに行く政治。」
2. **CREED / 政治信条** — 3つの原則(「必要な政治」を選ぶ／自己責任論に依存しない／善意頼みを許さない)
3. **PROFILE / プロフィール** — 基本情報・現職・資格・キャリアタイムライン
4. **ORIGIN / 活動の原点** — 終活支援の現場で見た「制度のすき間」への問題意識
5. **POLICY / 重点政策** — 3本柱(都市型セーフティネット／防災レジリエンス／行財政運営)
6. **MEDIA / 取材・メディア掲載実績** — BATONZ、葬研、PR TIMES、週刊エコノミストなどの取材記事カード
7. **ACTIVITY / 活動報告** — Instagram・X へのリンクと埋め込みエリア
8. **JOIN / ご支援** — 後援会入会・ボランティア・寄付の3枚カード
9. **CONTACT / お問い合わせ** — メール、SNSリンク

## 動作確認

`index.html` をダブルクリックするだけでブラウザで表示できます。サーバーは不要です。

## 設計方針(重要)

### 公職選挙法への配慮
本サイトは **「政治活動」用のサイト** として作成しています。以下の表現は使用していません。

- ❌ 「立候補します」「立候補予定」
- ❌ 「投票」「票」「当選」「議席」
- ❌ 「区議会議員選挙○○」など、特定の選挙を想起させる表現

代わりに以下のような表現を使用しています。

- ✅ 「政治活動」「地域のために」「現場主義」
- ✅ 「考えていること、取り組みたいこと」(政策)
- ✅ 「声を聞きに行く」

選挙期間が近づき、正式に立候補届出が受理された後は、サイト全体を「選挙運動用サイト」として再構成する必要があります。

### 都民ファーストの会との関係
公認許可が下りていない現時点では、以下を**徹底して避けています**。

- ❌ 「都民ファーストの会公認」と書かない
- ❌ 都民ファーストの会のロゴを使用しない
- ❌ 都民ファーストの会の党名を本人の肩書として記載しない

ただし、デザインのトーン(緑系の配色)は都民ファーストの会のブランドカラーを意識しています。
**公認が下り次第、ヒーロー部の eyebrow テキストの隣に公認バッジを追加するなどの修正を行う想定です。**

## カスタマイズ手順

### 1. 写真の差し替え

`assets/` フォルダに、以下のファイル名で写真を配置すると、即座に置き換わります。

| ファイル名 | 用途 | 推奨サイズ・形式 |
|---|---|---|
| `assets/portrait.jpg` | ヒーロー部の顔写真(縦長) | 800×1000px 程度、JPEG |
| `assets/profile.jpg` | プロフィール欄の活動写真(正方形) | 800×800px 程度、JPEG |

差し替え後、`index.html` の以下を `.svg` から `.jpg` に変更してください。

```html
<!-- ヒーロー部 -->
<img src="assets/portrait.svg" alt="田中せいじ" />
                       ↑ portrait.jpg に変更

<!-- プロフィール部 -->
<img src="assets/profile.svg" alt="田中せいじ 活動写真" />
                       ↑ profile.jpg に変更
```

### 2. キャリアタイムラインの修正

`<ul class="timeline">` 内の `<li>` を編集します。`class="hi"` を付けると節目の項目としてオレンジのマーカーが付きます。

```html
<li class="hi">
  <span class="year">2010</span>
  <b>見出しテキスト</b>
  <span class="sub">補足説明</span>
</li>
```

### 3. 重点政策の調整

`<section id="policy">` 内の `<article class="policy-card">` ブロックを編集します。

各カードは「リード文 + 3つの sub-point」で構成されています。

```html
<div class="point">
  <b>サブテーマ見出し</b>
  <span>説明文</span>
</div>
```

### 4. メディア掲載カードの追加・更新

`<section id="media">` 内の `.media-grid` に、以下の形式でカードを追加していきます。

```html
<a class="media-card" href="【記事URL】" target="_blank" rel="noopener">
  <div class="media-head">
    <span class="media-name">【媒体名】</span>
    <span class="media-tag interview">INTERVIEW</span>   <!-- またはrelease -->
    <span class="media-date">2025.04</span>
  </div>
  <h3>【記事タイトル】</h3>
  <span class="ex">記事を読む</span>
</a>
```

タグの種類:
- `interview` — オレンジ系。インタビュー・取材記事用
- `release` — グリーン系。プレスリリース・業務連携・サービス紹介用

新しいメディア掲載があるたびに、上記カードを追加するだけで増やせます。

### 5. 後援会連絡先の更新

`<section id="contact">` 内の `.contact-info` を編集します。

```html
<dl>
  <dt>事務所</dt><dd>田中せいじ後援会</dd>
  <dt>所在地</dt><dd>東京都荒川区○○ 1-2-3</dd>
  <dt>メール</dt><dd><a href="mailto:info@tanaka-seiji.jp">info@tanaka-seiji.jp</a></dd>
  <dt>電話</dt><dd>03-XXXX-XXXX</dd>
  ...
</dl>
```

※ 履歴書記載の携帯番号(090-1095-9623)は個人番号のため、公開用サイトには載せていません。
事務所固定電話が決まり次第、差し替えを推奨します。

### 6. Instagram投稿の埋め込み

「ACTIVITY」セクションの `.ig-embed-note` の下に、Instagramの投稿埋め込みコードを貼り付けることで、最新投稿を表示できます。

各投稿ページの「・・・」メニュー → 「埋め込み」からHTMLコードを取得できます。

### 7. YouTube動画の埋め込み

YouTubeチャンネル開設後、SNSグリッドにカードを追加するか、ACTIVITYセクションに `<iframe>` を追加します。

```html
<a class="sns-card" href="【YouTubeチャンネルURL】" target="_blank" rel="noopener">
  <div class="sns-icon yt" aria-hidden="true">
    <svg width="28" height="28" viewBox="0 0 24 24" fill="currentColor">
      <path d="M23.498 6.186a3.016 3.016 0 0 0-2.122-2.136C19.505 3.545 12 3.545 12 3.545s-7.505 0-9.377.505A3.017 3.017 0 0 0 .502 6.186C0 8.07 0 12 0 12s0 3.93.502 5.814a3.016 3.016 0 0 0 2.122 2.136c1.871.505 9.376.505 9.376.505s7.505 0 9.377-.505a3.015 3.015 0 0 0 2.122-2.136C24 15.93 24 12 24 12s0-3.93-.502-5.814zM9.545 15.568V8.432L15.818 12l-6.273 3.568z"/>
```

### 8. カラーテーマの調整

`index.html` 冒頭の `<style>` 内、`:root` セクションのCSS変数を変更してください。

```css
--green-700:#006a3a;   /* メインの緑(都民ファースト緑をベース) */
--warm:#f0a93a;        /* アクセントのオレンジ(温かみ) */
--paper:#fbfaf5;       /* 背景のオフホワイト */
```

## 公開・運用

### ホスティング(無料の選択肢)
- **Cloudflare Pages** ・・・ 推奨。ドメイン接続も簡単。
- **Netlify** ・・・ ドラッグ&ドロップで公開可能。
- **GitHub Pages** ・・・ GitHub アカウントが必要。
- **Xserver / さくらインターネット** ・・・ 独自ドメインで本格運用する場合。

### 推奨ドメイン
`tanaka-seiji.jp` (履歴書記載のメールアドレスとも整合)

## 公認決定後の更新ポイント

「都民ファーストの会」の公認が正式に決定した場合、以下の修正を行ってください。

1. ヒーロー部に公認ロゴ(都民ファーストの会から提供されるもの)を追加
2. メタディスクリプションに「都民ファーストの会公認」を追記
3. プロフィール表に「所属政党」行を追加

## 立候補届出後の更新ポイント

選挙運動期間に入った場合、本サイトは「選挙運動用サイト」となります。

1. 選挙運動用ホームページである旨の表示が必要
2. 電子メールアドレスの表示が法律上必須(info@tanaka-seiji.jp で対応済)
3. 公示日(告示日)以前は「投票」「当選」等の表現は引き続き使用不可
4. 公示日以降は、選挙運動用の表現に切り替え可能

詳細は総務省「インターネット選挙運動の解禁に関するQ&A」をご確認ください。
